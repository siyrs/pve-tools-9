### ✨ 主要特性

* 增加UPS信息
* 修复多个M2信息获取
* 修复多个HDD信息获取


#### 安装使用

```bash
# 1. 下载脚本
wget https://raw.githubusercontent.com/siyrs/pve-tools-9/main/PVE-Tools.sh

# 2. 添加执行权限
chmod +x PVE-Tools.sh

# 3. 运行脚本
sudo ./PVE-Tools.sh
```
